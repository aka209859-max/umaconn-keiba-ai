# NAR-SI3.0 指数分布分析レポート

**作成日**: 2026-01-10  
**作成者**: AI戦略家（NAR-AI-YOSO開発チーム）  
**対象期間**: 2016年～2025年  
**分析データ件数**: 66,685件

---

## 📊 エグゼクティブサマリー

### **分析目的**
- NAR-SI3.0の4指数（テン指数、上がり指数、位置指数、ペース指数）について、実装版ロジック（馬場差・不利・枠順・斤量・ペース補正を含む完全版）を用いて計算
- 5刻み・10刻みの区間別に件数、的中率、3着以内率を算出
- 正規化前の分布特性を把握し、今後の予測精度向上につなげる

### **重要な発見**
1. **位置指数**: 指数が高いほど的中率・3着以内率が明確に上昇（90~100区間で的中率39.4%）
2. **上がり指数**: 0~10区間で的中率27.1%と顕著な効果（プラス領域で大幅向上）
3. **ペース指数**: 0~10区間で的中率17.2%（プラス領域で効果的）
4. **テン指数**: 全体の92%が-10~0区間に集中（張り付き問題あり）

## 🎯 総合評価と推奨事項

### **指数の有効性ランキング**

| 順位 | 指数名 | 最大的中率 | 最大3着以内率 | 評価 |
|:---:|:---|---:|---:|:---|
| 🥇 1位 | **位置指数** | 39.92% | 71.37% | 最も信頼性が高い |
| 🥈 2位 | **上がり指数** | 27.09% | 57.85% | プラス領域で強力 |
| 🥉 3位 | **ペース指数** | 27.26% | 58.76% | プラス領域で有効 |
| 4位 | テン指数 | 13.33% | 40.00% | 張り付き問題あり |

### **推奨買い目条件（単勝）**

#### **最優先条件（的中率30%以上）**
```
位置指数 >= 85 AND (上がり指数 > 0 OR ペース指数 >= 5)
```

#### **高確率条件（的中率20%以上）**
```
位置指数 >= 80 AND 上がり指数 > 0
OR
位置指数 >= 90
OR
ペース指数 >= 5
```

#### **基準条件（的中率15%以上）**
```
位置指数 >= 70 AND (上がり指数 > 0 OR ペース指数 > 0)
OR
位置指数 >= 80
```

### **複勝推奨条件（3着以内率60%以上）**
```
位置指数 >= 80
OR
上がり指数 > 0 AND 位置指数 >= 70
OR
ペース指数 >= 5 AND 位置指数 >= 60
```
